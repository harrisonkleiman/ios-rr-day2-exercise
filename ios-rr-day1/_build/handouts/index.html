Required Reading: Day 1

Instructions

If you have not already, please download the Intro to App Development with Swift from the Apple Book Store: https://books.apple.com/us/book/intro-to-app-development-with-swift/id1118575552.

Please note, there are files to download in the Getting Started section. The can also be downloaded using the download button attached to this exercise.

  1. Please read the Introduction and chapters 1-4, as well as chapter 8.

  2. Complete the quiz at the end of each chapter

  3. Complete the corresponding Playground file for each chapter

  4. Complete the challenges and experiments for each chapter

  5. Apply what you learned from these sections by trying the recommended activities in the book.
